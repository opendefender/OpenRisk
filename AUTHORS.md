 How to Be Listed Here

We appreciate all contributions! To be added to this file:

. Submit a Pull Request with meaningful contributions
. After your PR is merged, you can add yourself to this file (or we'll add you)
. Include your:
   - Name or preferred identifier
   - GitHub username
   - Brief description of contributions (optional)
