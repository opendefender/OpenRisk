 KEYBOARD SHORTCUTS - Quick Reference

 Global Shortcuts (Anywhere)

| Key | Action |
|-----|--------|
| Ctrl+K / K |  Open global search |
| Ctrl+N / N |  Create new risk |
| Esc |  Close modal/dialog |

 Search Navigation (In Search)

| Key | Action |
|-----|--------|
| ↑ Arrow |  Previous result |
| ↓ Arrow |  Next result |
| Enter |  Select result |
| Esc |  Close search |

 Form Navigation (In Modal)

| Key | Action |
|-----|--------|
| Tab | → Next field |
| Shift+Tab | ← Previous field |
| Enter |  Submit form |
| Esc |  Cancel/Close |
| Space |  Toggle checkbox |

 Platform Notes

- Mac: Use  (Command) instead of Ctrl
- Windows/Linux: Use Ctrl key
- Mobile: Shortcuts not available (use buttons)

 Pro Tips

 Power Moves:
- Ctrl+K → Type → Arrow Down → Enter = Fast risk lookup
- Ctrl+N → Tab through fields → Ctrl+Enter = Quick create
- Esc = Safe cancel (no data loss in modal)
